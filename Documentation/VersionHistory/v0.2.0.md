# v0.2.0 - 캐시 관리 UI, Firebase 배포, 투명 배경 지원

**릴리스 날짜**: 2026-02-10

## 주요 변경사항

### 1. WebGL 투명 배경 지원

Unity WebGL 카메라 배경을 투명(alpha=0)으로 설정하여 Flutter 쪽에서 배경을 자유롭게 제어할 수 있도록 했습니다.

- 카메라 Clear Flags를 Solid Color + alpha 0으로 설정
- `ChangeBackground(string colorHex)` API로 Flutter에서 배경색 동적 변경 지원

### 2. 캐시 정보 UI 추가

FluentT SDK의 캐시 기능을 Flutter UI에서 확인하고 관리할 수 있도록 했습니다.

- `GetCacheInfo` → 캐시 개수 및 캐시된 ID 목록 조회
- `ClearCache` → 전체 캐시 초기화
- `FH_OnCacheInfo` 콜백으로 결과 전달

### 3. Firebase Hosting 배포

Flutter 웹 앱을 Firebase Hosting으로 배포할 수 있도록 설정했습니다.

- `firebase.json` 설정 (Cache-Control 헤더 포함)
- 배포 URL: `https://firsthabittalkmotionwebgl.web.app`
- 배포 가이드 문서 (`FIREBASE_DEPLOY.md`) 작성

### 4. Flutter 테스트앱 개선

- 전체 UI 리팩토링 및 사용성 개선
- Unity 설정 정리

### 5. 렌더링 품질 개선

- 메인 라이트 섀도우 맵 해상도를 2048에서 4096으로 증가
