# v0.1.0 - Flutter-Unity WebGL 브릿지 초기 구현

**릴리스 날짜**: 2026-02-06

## 주요 변경사항

### 1. 모노레포 구조 설정

Unity WebGL 프로젝트와 Flutter 테스트 앱을 하나의 레포지토리에서 관리하는 모노레포 구조를 설정했습니다.

- `unity/` - Unity WebGL 프로젝트
- `flutter_test_app/` - Flutter 웹 테스트 앱

### 2. Flutter-Unity WebGL 브릿지 구현

Unity WebGL 빌드를 Flutter 웹 앱의 iframe 안에 임베딩하고, 양방향 통신하는 브릿지를 구현했습니다.

- **C# → JS → Dart**: jslib 플러그인에서 `window.parent.FirsthabitBridge` 콜백 호출
- **Dart → JS → C#**: `SendMessage`를 통한 Unity 메서드 호출
- **지원 API**: `PrepareAudio`, `Play`, `Stop`, `ChangeBackground`, `ChangeAvatar` 스텁

### 3. Unity Wrapper HTML

Unity WebGL 로더를 감싸는 `unity_wrapper.html`을 통해 iframe 내에서 Unity 인스턴스를 관리합니다.
