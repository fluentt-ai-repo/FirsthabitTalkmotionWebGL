# 버전 히스토리

이 폴더는 FirsthabitTalkmotionWebGL 프로젝트의 각 버전별 변경사항을 문서화합니다.

## 버전 목록

- [v0.3.1](v0.3.1.md) - 2026-02-24 - 외부 개발자용 통합 가이드 문서, 콜백 등록 검증 (현재)
- [v0.3.0](v0.3.0.md) - 2026-02-24 - 아바타 런타임 교체, 자막 텍스트 전달, SDK 업데이트
- [v0.2.0](v0.2.0.md) - 2026-02-10 - 캐시 관리 UI, Firebase 배포, 투명 배경 지원
- [v0.1.0](v0.1.0.md) - 2026-02-06 - Flutter-Unity WebGL 브릿지 초기 구현

## 버전 번호 규칙

이 프로젝트는 [Semantic Versioning](https://semver.org/)을 따릅니다:
- **Major (X.0.0)**: 호환되지 않는 API 변경
- **Minor (0.X.0)**: 하위 호환 가능한 기능 추가
- **Patch (0.0.X)**: 하위 호환 가능한 버그 수정
