# v0.4.1 - Unity WebGL 빌드 파일 git 추적 추가

**릴리스 날짜**: 2026-02-25

## 주요 변경사항

### Unity WebGL 빌드 파일을 git 추적에 포함

배포 일관성을 확보하기 위해 기존에 `.gitignore`로 제외되던 Unity WebGL 빌드 결과물을 git에 포함시켰습니다.

- `flutter_test_app/.gitignore` 수정:
  - `web/unity_webgl/Build/`, `web/unity_webgl/TemplateData/`, `web/unity_webgl/index.html` 제외 규칙 제거
  - `build/` → `/build/`로 변경하여 Flutter 루트 빌드 디렉토리만 무시 (하위 `Build/` 폴더 영향 방지)

#### 추적 대상 파일 (총 ~92MB)

| 파일 | 크기 | 설명 |
|------|------|------|
| `unity_webgl.wasm` | 55MB | WebAssembly 바이너리 |
| `unity_webgl.data` | 36MB | 에셋 데이터 |
| `unity_webgl.framework.js` | 480KB | Unity 프레임워크 |
| `unity_webgl.loader.js` | 28KB | Unity 로더 |
| `index.html` | 2KB | Unity Wrapper HTML |
| `webmemd-icon.png` | 2KB | 아이콘 |

#### 참고사항

- `unity_webgl.wasm` (55MB)이 GitHub 권장 최대 크기(50MB)를 초과하지만, 제한(100MB) 이내
- 향후 빌드 크기가 증가할 경우 Git LFS 도입 검토 필요

## 수정 파일

- `flutter_test_app/.gitignore` — unity_webgl 제외 규칙 제거, build/ 패턴 수정
- `flutter_test_app/web/unity_webgl/` — 신규 추적: Build/, TemplateData/, index.html
