# v0.3.1 - 외부 개발자용 통합 가이드 문서, 콜백 등록 검증

**릴리스 날짜**: 2026-02-24

## 주요 변경사항

### 1. 외부 개발자용 Unity WebGL 통합 가이드 추가

Flutter Android 앱에서 InAppWebView를 통해 TalkMotion 아바타를 통합하는 종합 가이드를 작성했습니다.

- **INTEGRATION_GUIDE.md** (프로젝트 루트): 8개 섹션으로 구성
  - 아키텍처 다이어그램 및 통신 흐름 설명
  - Step-by-Step 통합 가이드 (HTML Wrapper, WebView 설정, 오디오 전달)
  - API Reference: Flutter→Unity 9개 메서드, Unity→Flutter 16개 콜백 상세 문서화
  - 완전한 HTML Wrapper 템플릿 및 Flutter Dart 샘플 코드 (`TalkmotionWebView` 위젯)
  - Troubleshooting (WASM 캐시, CORS, 오디오 포맷, WebView 설정)
  - 시퀀스 다이어그램 스타일의 전체 플로우 설명

- **InAppWebView 기반 아키텍처**: iframe 없이 WebView에서 Unity를 직접 로드하는 방식으로, jslib의 `window.parent` → `window` fallback을 활용

### 2. 콜백 등록 구조 검증 완료

외부 Flutter 개발자가 콜백을 자유롭게 사용할 수 있는지 검토했습니다.

- **기존 16개 콜백의 구독/해제/수정**: 완전히 자유 (InAppWebView의 `addJavaScriptHandler` 또는 Stream 패턴)
- **다중 리스너**: broadcast stream / 핸들러 내 분기로 가능
- **새 콜백 타입 추가**: jslib + C# 수정 필요 (허용 가능 수준으로 판단)
